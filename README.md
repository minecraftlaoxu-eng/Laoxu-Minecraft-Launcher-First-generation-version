# Laoxu-Minecraft-Launcher-First-generation-version
这是个初代版本




        Laoxu Minecraft Launcher 是个刚刚来的启动器，
        是老徐自己编程的启动器，它的本质是为了在 Android 端运行 Minecraft。
        
        同时，老徐还给启动器添加了基岩版的检测功能，
        能让大家在不翻找的情况下启动基岩版。
        
        如果你给 Laoxu Minecraft Launcher 赞助，
        老徐会很感激并把你写在赞助者列表里。
        
        老徐时刻会接受玩家们的热情！
